## :rocket: Getting started

### Requirements
- [Node.js](https://nodejs.org/en/)
- [Yarn](https://classic.yarnpkg.com/) or [npm](https://www.npmjs.com/)

### Clone the project
```bash
$ git clone https://github.com/mugasparetto/izymob.git && cd izymob
```

### Run these commands
```bash
# Install the dependencies
$ yarn

# Run the web client
$ yarn start
```
